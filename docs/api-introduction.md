# API Introduction

Welcome to the Ztake API. The API is RESTful and uses JSON over HTTPS. Use `https://api.ztake.in/` as the base URL.

## Examples

Production Base URL: `https://api.ztake.in/`

Support: `support@ztake.in`

## Notes

This documentation is an original, brand-specific reference for integrating with Ztake services. It is organized by product: Payouts, PG Collection (UPI), Verification Stack, BBPS, and more.
