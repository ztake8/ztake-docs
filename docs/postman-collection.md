# Postman Collection

A Postman collection for Ztake is included in the `/postman/` folder. Import it to quickly get started with example requests and environment variables.

## Examples

Included environment vars: BASE_URL, API_KEY, SAMPLE_TOKEN, SUPPORT_EMAIL

Import `/postman/postman_collection_ztake.json` into Postman and set API_KEY before sending requests.

## Notes

The collection groups requests into products: Ztake Payouts, PG Collection, Verification, BBPS, Webhooks, and Utility endpoints.
